## 🚀 Publicar en GitHub Pages (3 pasos)

### 1 · Crear repositorio
1. Abre [github.com](https://github.com) e inicia sesión
2. Clic en **"New repository"** → nombre: `justiberia` → **Public** → **Create**

### 2 · Subir `index.html`
**Opción web (sin instalar nada):**
1. En el repositorio recién creado, clic en **"uploading an existing file"**
2. Arrastra el archivo `index.html`
3. Clic en **"Commit changes"**

**Opción terminal:**
```bash
git init
git add index.html
git commit -m "Just Iberia web"
git remote add origin https://github.com/TU_USUARIO/justiberia.git
git push -u origin main
```

### 3 · Activar GitHub Pages
1. Ve a **Settings** → **Pages**
2. En *Source* selecciona **"Deploy from a branch"**
3. Branch: **main** · Carpeta: **/ (root)** → **Save**

### ✅ ¡Listo!
En 1–2 minutos tu web estará activa en:
```
https://TU_USUARIO.github.io/justiberia/
```
